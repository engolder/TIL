#### 환경 변수 설정
- 제어판\모든 제어판 항목\시스템 - 고급시스템 - 환경 변수
  - 시스템 변수(모든 사용자):
    - 새로 만들기: 변수 이름(N): JAVA_HOME --- 변수 값(V): C:\Program Files\Java\jdk1.7.0_09 (jdk 경로)
    - Path선택, 시스템 변수 편집: 변수 값(V): 맨 앞에 %JAVA_HOME%\bin; 추가
